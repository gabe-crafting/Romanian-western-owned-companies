# Romanian Western-Linked SRL Directory (Pilot)

Research shortlist of **43 active Romanian SRLs** with credible public evidence of Western ownership or leadership. This is **not legal due diligence**.

**Access date for all sources:** 2026-07-10

## Methodology and caveats

- Target size: active SRLs roughly in the **1–249 employee** range using latest filed financial year where available.
- **Micro / low-profile** means **1–10 employees** and usually limited public footprint. This is a neutral size/profile tag — not a claim of concealment or illegality.
- Three evidence labels:
  - **Likely Western-owned** — credible founder biography, company history, interview, or corporate disclosure supports Western ownership, but official shareholding is unavailable.
  - **Western-led, ownership unclear** — a Western founder/executive visibly leads the Romanian SRL, but ownership cannot be established from public sources.
  - **Verified Western-owned** — ONRC/shareholder data, foreign-capital registry markers, or an explicit authoritative ownership statement establishes ownership.
- Nationality is stated only when explicitly sourced. **English** is used only when a source explicitly identifies England; otherwise **British/UK**, or the person's sourced country.
- BRCC membership and British branding alone are **discovery signals**, not proof of ownership.
- Romania's beneficial-owner register generally requires legitimate interest and paid ONRC access, so many entries remain probabilistic.
